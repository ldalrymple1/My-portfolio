my projects.

EXPOSURE

INDIVUDUAL

Timeframe: 1 week

Determined to stretch myself, I built a Full-Stack application which was rooted in a Python and PostgreSQL back-end. Aimed at art enthusiasts, it allows users to browse upcoming art exhibitions in London. This was my first project working with Python and I was really satisfied with the finished product. Users can view the exhibitions in a traditional card view and map view; however, in due course I plan to add calendar view as well as other features.

SKILLSWAP

GROUP

Timeframe: 1 week

We were given a week to build a Full Stack MERN application. With only one weeks tuition in node.js and Express, I undertook the challenge of being responsible for developing the back-end. As aspiring developers, we decided to build a skill-swapping, community-led platform for junior developers. We built our own RESTful API as well as using a third-party API. This project taught me the benefits of favouring refactoring and bug fixing to ensure readable and maintainable code, imperative when sharing codebases.

FLASH FILMS

PAIR PROGRAMMING

Timeframe: 48hrs

As a pair we were built a React.js app using a third-party API. My partner and I shared a mutual interest in film, so we decided to use the MovieDB to build a website to help users choose a film. Working against the clock with just a 48-hour deadline, this project highlighted the importance of outlining and achieving a minimum viable product (MVP). I really enjoyed working in a pair and learnt some invaluable techniques from my partner.
